Getting started with SEPAL¶

Introduction¶

In this section of our documentation you’ll find the key requirements to use the platform. We’ll start from Register to SEPAL to the more advanced Exchange files with SEPAL. use the left panel to navigate in this section of the documentation. If you don’t find what you need please let us know using the GitHub issue tracker of this very website.

We will be happy to receive contributions from people that use this documentation, so if you find a typo, don’t hesitate to use the edit this page button on the right side of every documentation page to help us improve the way we present the platform!
